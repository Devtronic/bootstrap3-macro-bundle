# Page Header
## Description
`bootstrap.pageHeader(title, subtext)`

| Field        | Type       | Description                                             |
|:-------------|:----------:|:--------------------------------------------------------|
| title        | string     | The Heading                                             |
| subtext      | string     | A Subtext                                               |

## Example
```twig
{{ bootstrap.pageHeader('Example page header', 'Subtext for header') }}
```
